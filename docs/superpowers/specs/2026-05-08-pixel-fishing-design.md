## 1. 概述

一个基于 HTML5 + Phaser 3 的像素风钓鱼小游戏 Demo，采用 8-bit/16-bit 复古视觉风格，核心玩法为反应操作型——鱼上钩后通过按键操作收线。

**设计目标**：
- 模块化架构，便于后续功能扩展
- 完整可运行的核心游戏循环
- 程序化生成的像素素材，无需外部美术资源

---

## 2. 技术架构

### 2.1 技术栈
- **引擎**：Phaser 3.60+（CDN 引入）
- **文件格式**：单 HTML 文件，所有代码内联
- **素材**：程序化生成（Canvas 绘制）

### 2.2 核心模块

| 模块 | 职责 |

| SceneManager | 场景切换管理 |

| UISystem | HUD、结算面板 |
| AnimationSystem | 角色动画、粒子效果 |

## 3. 游戏流程

### 3.1 状态机

```
  [IDLE] ──点击────> [CASTING] ──动画完成──> [WAITING]
    ↑                                    │
    │                                    ↓
  结算                                 [FISH_ON] ←─检测咬钩
    ↑                                    │
    │                         ┌──────┐   │
    └──────游戏结束────────────│PULLING│←──┴──按键收线
                  └──────────> [SUCCESS] ──→ [IDLE]
```

### 3.2 状态说明

| 状态 | 行为 | 持续时间 |
|------|------|----------|
| IDLE | 等待玩家操作，显示抛竿提示 | 任意 |
| CASTING | 播放抛竿动画 | 0.5秒 |
| WAITING | 浮标漂浮动画，随机等待咬钩 | 2-6秒 |
| FISH_ON | 鱼上钩提示，力度条出现 | 瞬时 |
| PULLING | 按键收线，鱼拉扯 | 动态 |
| SUCCESS | 显示钓获鱼信息 | 2秒 |

## 4. 收线小游戏

### 4.1 力度条机制

```
┌─────────────────────────────────────┐
│  SAFE ZONE                          │
│  ├──────────────┤                  │
│  0%            50%          100%   │
│         ████ FISH ████             │
│                                     │
│  当前进度: ████████░░░░ 65%        │
└─────────────────────────────────────┘
```

### 4.2 规则
- 进度条自动向右推进
- 鱼在 SAFE ZONE 内随机左右拉扯
- **安全区内按键**：进度 +1%/帧
- **安全区外按键**：进度 -2%/帧
- **进度 ≥ 100%**：成功钓起！
- **进度 ≤ 0%**：鱼逃跑

## 6. 视觉规格

### 6.1 像素风格
- 分辨率：480x320（16:9 像素风格）
- 像素密度：2x（每像素实际显示 2x2 块）
- 渲染：NearestFilter（无抗锯齿）
